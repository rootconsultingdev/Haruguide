메인 페이지 구성 요소

① 과정 제안 필요

홈페이지를 통해 들어온 문의에 대한 확인 및 응대 조회 부분

② 전일 신규 고객

문의 확인 및 응대하면서 추가되는 고객 자료들

③ 전일 고객 문의

홈페이지로 들어온 문의 건 수

④ 전일 수주

최종 과정 제안 및 수주 시 관련 내용이 카운트되는 부분

⑤ 최근 과정 관리

최근 등록된 과정 표시
우선순위: 담당자 이름 일치 → 소속 부서 → 전체 최신순

⑥ 전체 공지사항

프로그램 사용에 필요한 내용 또는 공지할 사항들

⑦ 간이 일정표

일정표를 통해 본인의 일정을 확인할 수 있습니다

⑧ 내 정보 및 빠른 이동

정보 수정 및 각 플랫폼으로 이동하는 바로가기 버튼

사이드바 메뉴 구성

공지사항 고객관리 과정관리 제안 현황 회의실 예약 개인 일정표 교보재 관리 시스템

고객관리

고객 등록 · 조회 · 수정 방법

등록한 고객의 기업명, 부서, 이름, 직급, 연락처, 이메일을 확인할 수 있습니다.

1. 검색 가능한 항목

고객 성함 기업명 부서 직급 연락처 이메일 메모
▲ 고객 관리 목록 화면

2. 고객 등록 방법

  • '신규 고객 등록' 버튼을 클릭하여 정보를 입력합니다.
  • 필수 입력 항목 : 이름 기업명
⚠️
가능한 많은 정보를 입력해주시기를 부탁드립니다.
같은 고객의 중복 등록 방지를 위함입니다.

등록된 고객 수정 방법

  • 고객 목록에서 해당 고객을 선택하면 우측의 고객 등록 창으로 들어갑니다.
  • 수정 버튼을 눌러서 수정 시작, 수정 완료 버튼으로 정보를 저장합니다.
▲ 신규 고객 등록 버튼
5

과정 관리

과정 등록 · 조회 · 매출 등록 방법
▲ 과정 관리 목록 화면

과정 조회 가능한 내용 (요약)

기업명 과정명 루트 내부 담당자명 고객명 과정 카테고리 교육대상 프로그램명 사용 진단 수주 여부 진행 상태
▲ 과정 상세 조회 화면
📋
과정 상세 조회

각 과정을 클릭하면 과정의 상세 정보를 확인하실 수 있습니다.

이 때 '각 과정의 매출 내역' 또한 입력 및 조회가 가능합니다.
매출 등록 방법은 하단의 '과정의 매출을 등록하는 방법' 을 확인해주세요.

신규 과정 추가 방법

▲ 과정 등록 데모 (고객 관리 탭에서 접근)
➕
고객 관리 탭에서 과정을 추가할 수 있습니다. 오른쪽 상단의 과정 추가 버튼을 통하여 신규 과정 등록을 시작합니다.
▲ 과정 등록 상세 화면

입력 필수 정보

분류 항목
기본 정보 과정명, 카테고리, 진행 상태, 수주 여부
담당자 정보 과정을 진행한 부서, 과정의 담당자
기타 고객이 요청한 제안 마감일
* 정해져 있지 않은 경우 제안서 전달 예정일로 선택
ℹ️
진행 상태가 제안 중인 경우 6. 제안 현황 탭에서 확인할 수 있습니다.
제안이 완료된 경우도 제안 현황 탭에서 진행 상황을 확인하실 수 있습니다.
⚠️
미수주 시에는 "제안 완료 — 수주 여부 X" 표시를 해주시고,
[비고] 란에 미수주 사유를 꼭 적어주세요.
🗂️
(중요) 관련 자료 등록 시 NAS에 자동 동기화가 진행됩니다.
양식에 맞추어 자료를 등록해주세요.
▲ 관련 자료 등록 양식
▲ 자료 폴더 구조

📁 자료 자동 동기화 — 어떻게 진행되는가?

아래 단어가 파일명에 포함된 경우, 자동으로 해당 폴더에 저장됩니다.

강의안 견적서 교보재 운영자료 제안서 .mp4 → 영상 폴더
📌
여러 단어가 혼재된 경우 위 순서로 우선순위 적용.
예) 고객사명_과정명_프로그램 제안_견적서 → 견적서 폴더
해당 단어가 없는 경우 → 분류 폴더 없이 과정명 폴더 안에 저장됨

과정의 매출을 등록하는 방법

▲ 과정 상세 내역 → '매출 수정' 버튼 위치
💰
과정 상세 내역에서 우측 하단에 보이는 '매출 수정'을 클릭하여 진입합니다.
▲ 매출 등록 화면
📅
매출 정보가 없는 경우 '+ 월간 매출 추가' 버튼을 눌러 매출 발생 월을 등록해야 합니다.
매출 발생 월은 예상 매출 월에 맞추어 기입해주세요.

예시) 세금계산서가 10월에 발행될 예정이다. → 10월의 월간 매출 추가 등록
▲ 매출 추가 → 10월 클릭 화면
▲ 예상 매출 내역 작성 화면

매출 월이 생성되었다면 다음과 같은 예상 매출 내역을 작성하실 수 있습니다.
진행 부서와 담당자를 확인해주시고, 강의/FT 매출 · 과정 개발 매출 · 진단 매출 · 기타 매출 등을 입력해주세요.

여러 부서 간 매출 분배가 발생하였다면 매출 분배 버튼을 통하여 분배 작성을 할 수 있습니다.

▲ 매출 분배 작성 화면

예상 매출을 다 작성하셨다면 매출 저장을 눌러서 입력을 마무리해주세요.
입력된 매출은 추후 해당 과정의 상세 정보에서 확인하실 수 있습니다.

▲ 매출 저장 완료 화면
🔁
특정 교육과정들의 경우 같은 이름/내용의 과정이 1년 중 반복해서 발생하는 경우가 있습니다.
이런 경우를 위하여 각 월마다 매출을 반복해서 등록할 수 있게끔 만든 기능입니다.

제안 현황

각 부서 제안 현황 및 미수주 과정 조회

현재 각 부서들의 제안 현황 및 미수주 과정을 볼 수 있습니다.

▲ 제안 현황 전체 화면

각 카테고리를 오름차순/내림차순으로 정렬할 수 있으며,
우측 상단의 토글 버튼을 통하여 특정 부서의 제안 현황만 확인하는 것도 가능합니다.

🔍
통합 검색 기능
부서, 기업명, 과정명, 프로그램명 등의 검색어를 입력하여 진행사항을 검색할 수 있습니다.
▲ '기아' 검색 결과 예시

회의실 예약

회의실 및 온라인 미팅 도구 예약 방법
▲ 회의실 예약 캘린더 화면

예약 방법

  1. 우측 상단에서 예약할 카테고리 선택

예약 가능 카테고리

오너스 5층 스튜디오 12층 회의실 집중작업실 ZOOM Webex 교육용 MS 계정

개인 일정표

개인 일정 등록 및 제목 작성 규칙

등록 방법

▲ 개인 일정표 화면
  • 우상단의 일정 추가를 통해 일정 설정이 가능합니다.

일정 제목 작성 규칙 (중요)

📌
제목을 적으실 때에는 다음 순서를 꼭 지켜주시기 바랍니다.
[강사명, *외부 강사 진행 시](지역)(시간)(과정명)

괄호는 없어도 됩니다.

선택일정 일괄삭제 기능

사용 방법

  1. 일정표에서 삭제할 일정을 Ctrl + 클릭합니다.
    → 선택된 일정은 빨간색 테두리(아웃라인)로 표시됩니다.
  2. 여러명의 일정 (각각 다른 것)을 제거할 때,
    Ctrl + 클릭으로 원하는 일정들을 모두 선택합니다.
  3. 상단의 '선택일정 일괄삭제' 버튼을 클릭하여
    선택된 일정들을 한 번에 삭제합니다.
⚠️
일괄삭제는 되돌릴 수 없습니다.
선택된 일정을 다시 한번 확인하신 후 삭제를 진행해주세요.

교보재 관리 시스템

물품 조회 · 반출 신청 · 반납 방법

사용 방법 안내

🔐
고객관리시스템과 별도의 서비스이므로 로그인이 필요합니다.
계정 정보는 초기 시스템 정보와 같습니다. (계정명 / 비밀번호 1)

물품 리스트

▲ 물품 리스트 화면
  • 물품 리스트에는 현재 있는 전체 교보재 수량을 확인하실 수 있습니다.
  • 물품 개수가 변동되었거나 새로 등록되는 경우 물품 추가 및 수정을 통해 수정할 수 있습니다.

물품 반출

▲ 물품 반출 화면

물품 반출 탭에서 물건을 반출할 수 있습니다.

  1. 반출일과 반납 예정일 선택
  2. 반출 물품 추가 — 반출함에서 갯수 체크
  3. 반출 사유 작성
  4. 예약 확인 진행

나의 반출 현황

▲ 나의 반출 현황 화면
  • 사용자가 반출했던 물건들에 대하여 현황 파악 및 반납 진행을 하는 곳입니다.
📝
반납 시에는 원위치시켰는지, 실제 교보재를 어디에 두었는지 등 코멘트로 적어주시면 감사하겠습니다.
▲ 반출 현황 상세

물품 반출 현황 (전체)

▲ 전체 물품 반출 현황 화면
  • 반출하고자 하는 교보재가 현재 다른 곳에 반출되어 있을 때, 이를 확인할 수 있는 전체 유저 반출 현황 체크판입니다.
