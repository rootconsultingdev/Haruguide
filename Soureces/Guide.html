3. 공지사항
  • 공지사항에서는 관리자가 올린 공지사항을 게시판 형태로 확인하실 수 있습니다
4. 고객관리
  • 등록한 고객의 기업명, 부서, 이름, 직급, 연락처, 이메일을 확인할 수 있습니다.

    1. 고객 관리 검색 가능한 내용

    • 고객 성함, 기업명, 부서, 직급, 연락처, 이메일. 메모(고객 별 특성을 작성 및 저장 가능)

2. 고객 등록 방법

  • '신규 고객 등록' 버튼을 클릭하여 정보를 입력합니다.
  • 필수 입력 항목 : 이름, 기업명
  • **가능한 많은 정보를 입력해주시기를 부탁드립니다. ( 같은 고객의 중복등록 방지를 위함 )

등록된 고객 수정 방법

  • 고객 목록에서 해당 고객을 선택하면.
    우측의 고객 등록 창으로 들어갑니다.
  • 수정 버튼을 눌러서 수정 시작,
    수정 완료 버튼을 통해 정보를 수정합니다.

5. 과정 관리

과정 조회 가능한 내용 (요약)

  • 기업명, 과정명, 루트 내부 담당자명, 고객명, 과정 카테고리, 교육대상, 프로그램명, 사용 진단.
  • 과정이 수주되었는지. 진행 상태는 어떤지도 함께 조회 가능합니다.

과정 조회 가능한 내용 (상세)

  • 각 과정을 클릭하였을 때,
    과정의 상세 정보를 확인하실 수 있습니다.
    이 때, ‘각 과정의 매출 내역’
    또한 입력 및 조회가 가능합니다.
    매출 등록과 관련된 내용은, 하단의 ‘과정의 매출을 등록하는 방법’ 을 확인해주세요.

신규 과정 추가 방법
  • 고객 관리 탭에서 과정을 추가할 수 있습니다.
    오른쪽 상단에 위치한 과정 추가 버튼을 통하여 신규 과정 등록을 시작할 수 있습니다.
  • 과정 등록에서는 다음과 같은 정보들을 입력해주셔야 합니다.
    기본 정보 : 과정명, 카테고리, 진행 상태, 수주 여부
    담당자 정보 : 과정을 진행한 부서, 과정의 담당자
    *고객이 요청한 제안 마감일(정해져 있지 않은 경우, 제안서 전달 예정일로 선택해주세요.)
  • 진행 상태 탭에서 제안 중인 경우에는, 6. 제안 현황 탭에서 확인할 수 있습니다.
    제안이 완료된 경우 또한 6. 제안 현황 탭에서 진행 상황을 확인하실 수 있습니다.
  • 미수주 시에는, 제안 완료 - 수주 여부 X 표시를 해 주시고.
    [비고] 란에 미수주 사유를 꼭 적어주세요.
  • (중요) 관련 자료 등록 시 NAS에 자동 동기화가 진행됩니다.
    양식에 맞추어 자료를 등록해주세요.

    ※자료 자동 동기화는 어떻게 진행되는가?
    ’강의안’ , ’견적서’, ’교보재’, ’운영자료’, ’제안서’
    각 단어가 포함된 경우, 자동으로 폴더에 저장됩니다.
    * 영상의 경우 ‘.mp4’ 파일이 등록될 경우, 영상 폴더로 자동으로 이동됩니다.

    ex) 고객사명_과정명_강의안 → 강의안 폴더
    여러 단어가 혼재된 경우 위 순서대로 우선순위 적용.
    고객사명_과정명_ 프로그램 제안_견적서
    → 견적서가 우선순위이니 견적서 폴더로

    만약 저 단어들이 들어있지 않다면?
    각 분류 폴더에는 들어가지 않은 상태로 과정명 폴더 안에 위치하게 됩니다.

과정의 매출을 등록하는 방법
  • 과정 상세 내역에서, 우측 하단에 보이는 ‘매출 수정’ 을 클릭하여 진입합니다.
  • 기본적으로 아무 매출정보도 등록되지 않은 경우. ‘ + 월간 매출 추가 ‘ 버튼을 눌러 매출 발생 월을 등록해주셔야 합니다.
    매출 발생 월은 예상 매출 월에 맞추어 기입해주시면 됩니다.
    예시) 세금계산서가 10월에 발행될 예정이다. → 10월의 월간 매출 추가 등록.
    • 매출 추가 → 10월 을 클릭한 모습입니다.

    매출 월이 생성되었다면, 다음과 같은 예상 매출 내역을 작성하실 수 있습니다.
    진행 부서와 담당자를 확인해주시고, 강의/FT 매출, 과정 개발 매출, 진단 매출, 기타 매출 등을 적어주세요.

    만약 여러 부서간 매출 분배가 발생하였다면, 매출 분배 버튼을 통하여, 분배 작성을 할 수 있습니다.
    * 매출 분배를 클릭하여 분배 매출을 만든 예시입니다.

    예상 매출을 다 작성하셨다면. 매출 저장을 눌러서 입력을 마무리해주세요.
    입력 된 매출은 추후 해당 과정의 상세 정보에서 확인하실 수 있습니다.

    *특정 교육과정들의 경우에 한하여, 같은 이름/내용의 과정이 1년 중 반복해서 발생하는 경우가 있습니다.
    그런 경우를 위하여, 같은 내용의 과정을 반복해서 등록하는 피로감을 방지하기 위하여 각 월마다 매출을 반복해서 등록할 수 있게끔 만든 기능입니다.

6. 제안 현황

현재 각 부서들의 제안 현황 및 미수주 과정을 볼 수 있습니다.

각 카테고리를 오름차순/내림차순으로 정렬을 확인하실 수 있으며,
우측 상단의 토글 버튼을 통하여 특정 부서의 제안 현황만 확인하시는 것 또한 가능합니다.

상단의 통합 검색의 경우
부서, 기업명, 과정명, 프로그램 명 등의 검색어를 입력하여 진행사항을 검색해 볼 수 있습니다.
*기아 를 검색한 경우

7. 회의실 예약

등록 방법

  • 우측 상단에서 어느 회의실을 예약할지 카테고리를 선택합니다.
  • 선택 가능한 카테고리 목록 :
    오너스 5층 스튜디오, 12층 회의실 및 집중작업실, ZOOM, Webex, 교육용 MS 계정

- 카테고리 선택 후 캘린더에서 원하는 날짜와 시간을 선택합니다.

- 회의 목적, 참석자, 사용 시간을 입력합니다.

8. 개인 일정표

등록 방법

  • 우상단의 일정 추가를 통해 일정 설정이 가능합니다.
  • (중요) 제목을 적으실 때에는, 다음과 같은 순서를 꼭 지켜주시기 바랍니다. 하기 참조, 괄호 없어도 됨.

[강사명, *외부 강사 진행 시](지역)(시간)(과정명)
위 순서로 일정의 제목을 적어주세요.

9. 매출 통계

매출 통계 부분에서는, 입력된 과정들의 매출 데이터를 확인해볼 수 있습니다.

  • 매출 통계는 월간, 분기, 반기, 연간으로 되어 있으며. 해당 시트들은 엑셀로 출력 또한 가능합니다.
10. 교보재 관리 시스템

사용 방법

  • 고객관리시스템과 별도의 서비스이므로, 로그인이 필요합니다.
  • 계정 정보는 초기 시스템 정보와 같습니다. ( 계정명 / 비밀번호 1 )

물품 리스트

  • 물품 리스트에는 현재 있는 전체 교보재 수량을 확인하실 수 있습니다.
  • 물품 갯수가 변동되었거나, 새로 등록되는 경우 물품 추가 및 수정을 통해 이를 수정할 수 있습니다.

물품 반출

  • 물품 반출 탭에서 물건을 반출할 수 있습니다.
  1. 반출일과 반납 예정일 선택
  1. 반출 물품 추가. 반출함에서 갯수 체크
  1. 반출 사유 작성
  1. 예약 확인 진행

나의 반출 현황

  • 사용자가 반출했던 물건들에 대하여, 현황 파악 및 반납 진행을 하는 곳입니다.
  • 반납 시에는 원위치시켰는지. 실제 교보재를 어디에 두었는지 등 코멘트로 적어주시면 감사하겠습니다.

물품 반출 현황

  • 반출하고자 하는 교보재가 현재 다른 곳에 반출되어 있을 때, 이를 확인할 수 있는 전체 유저에 대한 반출 현황 체크 판입니다.

11. NAS 자동 동기화 확인

과정 등록 시 입력한 자료는, 고객관리 시스템 내에서도 확인할 수 있지만, NAS로도 확인 및 다운로드가 가능합니다.
